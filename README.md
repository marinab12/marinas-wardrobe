# Marina's Wardrobe

## Overview
Marina's Wardrobe is a web application designed to help users browse and select clothing items for different seasons. The application features a visually appealing layout that allows users to switch between summer and winter clothing categories seamlessly.

## Project Structure
```
marinas-wardrobe
├── src
│   ├── web.html          # HTML structure of the web application
│   ├── app.js            # JavaScript logic for handling user interactions
│   └── styles.css        # CSS styles for the application
├── assets
│   ├── summer            # Directory containing summer clothing images
│   │   ├── head          # Images for summer headwear
│   │   ├── arriba        # Images for summer tops
│   │   ├── chaqueta      # Images for summer jackets
│   │   ├── vestido       # Images for summer dresses
│   │   ├── abajo         # Images for summer bottoms
│   │   ├── zapatos       # Images for summer shoes
│   │   ├── bolso         # Images for summer bags
│   │   └── accesorio     # Images for summer accessories
│   └── winter            # Directory containing winter clothing images
│       ├── head          # Images for winter headwear
│       ├── arriba        # Images for winter tops
│       ├── chaqueta      # Images for winter jackets
│       ├── vestido       # Images for winter dresses
│       ├── abajo         # Images for winter bottoms
│       ├── zapatos       # Images for winter shoes
│       ├── bolso         # Images for winter bags
│       └── accesorio     # Images for winter accessories
├── data
│   └── items.manifest.json # Manifest file containing item metadata
├── .vscode
│   └── settings.json      # VS Code specific settings
└── README.md              # Project documentation
```

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the project in your preferred code editor.
3. Open `src/web.html` in a web browser to view the application.
4. Ensure that the `assets` directory contains the necessary images for both summer and winter categories.

## Usage Guidelines
- Use the buttons or interface elements to switch between summer and winter clothing options.
- Click on the images to select items, which will update the display accordingly.
- The application is designed to be responsive and should work on various devices.
